• Bold Tag
    • Bold tag is used to make text look bold.
    • Used to prioritize any specific word
  • Italic Tag
    • Italic tag is used to make text look slightly bent.
    • Not used nowadays
    • Used to showcase author name, address.
  • Strong Tag
    • Strong tag is almost similar to Bold tag.
    • When we command someone what language we use, same is with the strong tag. Example: Sit Down!
  • Emphasis Tag
    • Emphasis tag is almost similar to Italic tag.
    • When we express someone what language we use, same is with the Emphasis tag.Example : Hooray! Everything happened fine.
