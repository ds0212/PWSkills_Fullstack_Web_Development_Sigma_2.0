1. `p`: Defines a paragraph.
  2. `h1`, `h2`, `h3`, ... `h6`: Define headings with varying levels of importance.
  3. `a`: Creates hyperlinks.
  4. `img`: Embeds images.
  5. `ul`: Defines an unordered list.
  6. `ol`: Defines an ordered list.
  7. `li`: Represents a list item within a list.
  8. `div`: Defines a division or a container for other elements.
  9. `span`: Defines an inline container for a small piece of content or text.
  10. `table`, `tr`, `td`: Used for creating tables and table rows and cells.
